## 빠른 판정 기준

- `통과`: 모든 필수 텍스트가 배경과 구분되어 읽힌다.
- `수정 필요`: 하나 이상의 텍스트가 배경에 묻히거나 잘린다.
- `확인 불가`: 데이터나 권한 부족으로 화면을 열 수 없다.
- `미확인`: 아직 검수하지 않았다.

## 완료 선언 조건

- 모든 필수 화면이 데스크톱 기본, 데스크톱 다크, 모바일 기본, 모바일 다크에서 `통과`여야 한다.
- `수정 필요`와 `미확인` 항목이 없어야 한다.
- `확인 불가` 항목은 사유와 후속 확인 계획이 있어야 한다.
- 빌드 성공만으로는 완료가 아니며, 실제 화면 증거가 있어야 한다.
