# UI 대비 검수 대상 화면 인벤토리

이 문서는 TravelLedger에서 배경과 글자 대비를 검수할 때 반드시 열어야 하는 화면 목록이다.
실제 완료 판정은 이 목록의 각 화면을 기본 테마, 다크 테마, 모바일, 데스크톱에서 확인한 뒤에만 가능하다.

## 검수 조건

| 조건 | 값 |
| --- | --- |
| 데스크톱 viewport | 1440 x 900 |
| 모바일 viewport | 390 x 844 |
| 테마 | 기본 테마, 다크 테마 |
| 권한 | 일반 사용자, 관리자 |
| 상태 | 기본, hover, focus, active, selected, disabled, readonly, loading, empty, error |

## 인증/계정

| 화면 | 권한 | 주요 확인 요소 |
| --- | --- | --- |
| 로그인 | 비로그인 | ID/비밀번호 입력, 버튼, 오류 메시지 |
| 회원가입 | 비로그인 | 초대 코드, PIN 입력, 도움말, 검증 메시지 |
| 초대 링크 로그인 | 비로그인 | 초대 안내, 계정 생성 안내, 버튼 |
| 프로필 | 일반 사용자 | 개인정보, 2차 PIN, 데이터 export, 모달 |
| 관리자 추가 인증 | 관리자 | 3차 비밀번호 모달, placeholder, 확인 버튼 |

## 완료 기준

- 이 문서의 모든 필수 화면을 기본 테마와 다크 테마에서 확인한다.
- 각 화면을 데스크톱과 모바일 viewport에서 확인한다.
- 문제 발견 시 스크린샷과 조건을 `docs/ui_contrast_audit_evidence_current.md`에 기록한다.
- `미확인` 또는 `수정 필요` 항목이 남아 있으면 목표 완료로 보지 않는다.
